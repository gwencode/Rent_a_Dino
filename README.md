# Name
Rent a dino 

## Description 

Rent a dino is a small clone of Airbnb, but for renting dinosaurs.

It is a basic CRUD web application. 

## Technologies used 

Ruby on Rails, PostgreSQL, HTML, CSS, Bootstrap, Figma, Webpack, Heroku
